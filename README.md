# Vite + Flask

To setup on python:
```
python -m venv venv

(windows)
.\venv\Scripts\activate.bat

(linux)
source venv/bin/activate

pip install -r requirements.txt
```
Technologies used:
- flask
- flask-cors
- openai
- matplotlib
- Java Script
- React
